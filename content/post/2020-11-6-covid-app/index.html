Like all of my posts, this is a work in progress. The Covid Tracker App currently allows one to visualize the number of new cases and new deaths over time. The default States are California, Hawaii and Florida but you can add or remove States using the selector box in the sidebar. The accompanying maps use color to show the total cases and total deaths per million people by State. The data is from Johns Hopkins University and is updated daily. You can explore the app here embedded in this post or, better, explore it using the link above. The app is hosted on shinyapps.io and my code is available on Github
